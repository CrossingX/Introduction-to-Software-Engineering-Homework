# 贪吃蛇实验报告
## 一、程序伪代码
```
输出字符矩阵
	WHILE not 游戏结束 DO
		ch＝等待输入
		CASE ch DO
		‘A’:左前进一步，break 
		‘D’:右前进一步，break    
		‘W’:上前进一步，break    
		‘S’:下前进一步，break    
		END CASE
		输出字符矩阵
	END WHILE
	输出 Game Over!!!
```

## 二、源代码实现及细节
snake_move.c只需按照伪代码来实现即可，snake_eat.c是在snake_move.c的基础上进行添加函数，不断试验的操作。
<br/>我用的方法是链式队列，队首即为蛇尾，队尾为蛇首，根据先入先出原则(FIFO),贪吃蛇每走一步，就出队一次，入队一次，然后新的队尾变成蛇头，新的队首变成蛇尾。
<br/>snake_eat.c中，贪吃蛇吃了食物后要变长，所以此时不用出队，只需将食物坐标入队即可。
